# Reto 2 — Notebooks

## Objetivo
Construir un sistema de inteligencia competitiva que soporte decisiones de pricing, operaciones y estrategia por zona, a partir de datos públicos de competidores.

## Estado actual

**Reto 2 no está en desarrollo activo todavía.**

El Reto 1 (Operational Intelligence) está en progreso. El Reto 2 comenzará una vez que el Reto 1 esté completo.

## Estructura preparada

| Notebook | Propósito previsto |
|---|---|
| `00_reto2_data_acquisition.ipynb` | Adquisición y limpieza de datos públicos de competidores |
| `10_reto2_eda.ipynb` | EDA de datos competitivos |
| `20_reto2_analysis.ipynb` | Análisis comparativo y scoring por zona |
| `30_reto2_intelligence.ipynb` | Alertas y recomendaciones de inteligencia competitiva |

## Qué se necesitará

- Definir fuentes de datos públicos de competidores.
- Diseñar estrategia de adquisición (scraping, APIs públicas, datos abiertos).
- Definir dimensiones de comparación relevantes por zona.
- Diseñar cómo integrar inteligencia competitiva con el sistema de Reto 1.
